# UCS 761 -- Deep Learning Lab Assignments

This repository contains the laboratory exercises completed as part of
the **UCS 761 -- Deep Learning** course. The assignments focus on
implementing core neural network models and machine learning algorithms
from scratch to understand the internal workings of perceptrons,
regression models, and multilayer neural networks.

The implementations are written in **Python** using **NumPy, Pandas,
Matplotlib, and Scikit‑learn**, and executed using **Google Colab /
Jupyter Notebook**.

# Lab Exercises

## Lab Exercise 1 -- Perceptron for Logic Gates

Design and implement a **Perceptron model** to learn the following
logical operations:

-   AND Gate\
-   OR Gate\
-   NAND Gate\
-   NOR Gate

### Concepts Covered

-   Perceptron model\
-   Linear decision boundaries\
-   Perceptron learning rule\
-   Binary classification

### Reference

https://www.enjoyalgorithms.com/blog/design-perceptron-to-learn-and-or-xor-gates

------------------------------------------------------------------------

## Lab Exercise 2 -- Logistic Regression using Single Layer Perceptron

Implementation of **Logistic Regression using a Single Layer Perceptron
Neural Network (SLPNN)** to classify glass types.

### Dataset

Glass Identification Dataset

Dataset Link\
https://www.kaggle.com/datasets/uciml/glass

### Reference

https://medium.com/analytics-vidhya/logistic-regression-using-single-layer-perceptron-neural-network-slpnn-31757c792d5d

### Concepts Covered

-   Logistic regression\
-   Sigmoid activation function\
-   Cross-entropy loss\
-   Gradient descent\
-   Multiclass classification using one-hot encoding

------------------------------------------------------------------------

## Lab Exercise 3 -- Multiple Linear Regression using Perceptron

Implementation of **Multiple Linear Regression using a perceptron-based
model** trained using gradient descent.

### Dataset

Multiple Linear Regression Dataset

Dataset Link\
https://www.kaggle.com/datasets/hussainnasirkhan/multiple-linear-regression-dataset

### Concepts Covered

-   Linear regression\
-   Mean Squared Error (MSE)\
-   Gradient descent optimization\
-   Weight and bias updates\
-   Regression using neural network formulation

------------------------------------------------------------------------

## Lab Exercise 4 -- Multilayer Perceptron with Backpropagation

Implementation of a **Multilayer Perceptron (MLP)** trained using the
**Backpropagation algorithm** to predict the age of abalone from
physical measurements.

### Dataset

Abalone Dataset

Dataset Link\
https://archive.ics.uci.edu/dataset/1/abalone

### Concepts Covered

-   Multilayer neural networks\
-   Hidden layers\
-   Forward propagation\
-   Backpropagation algorithm\
-   Regression using neural networks\
-   Mean Squared Error loss
